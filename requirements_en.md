# Requirements Guide (English)

## Installation

```bash
pip install -r requirements.txt
```

## Main Dependencies

- `torch` / `torchvision` / `torchaudio`: deep learning training and inference
- `timm`: backbone and model utility components
- `einops`: tensor rearrangement
- `numpy` / `scipy` / `scikit-image`: numerical computing and image metrics
- `opencv-python` / `pillow` / `matplotlib`: image processing and visualization
- `pandas`: result aggregation and export
- `tqdm`: progress bars
- `tensorboardX`: training log visualization

## Notes

- `requirements.txt` is the only file used by `pip` for installation.
- This file is documentation only.
