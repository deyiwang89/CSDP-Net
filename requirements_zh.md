# 依赖说明（中文）

## 安装方式

```bash
pip install -r requirements.txt
```

## 核心依赖

- `torch` / `torchvision` / `torchaudio`：深度学习训练与推理
- `timm`：部分网络模块依赖
- `einops`：张量重排
- `numpy` / `scipy` / `scikit-image`：数值计算与图像指标
- `opencv-python` / `pillow` / `matplotlib`：图像处理与可视化
- `pandas`：结果统计与导出
- `tqdm`：进度条
- `tensorboardX`：训练日志可视化

## 说明

- `requirements.txt` 是唯一安装入口。
- 本文件仅用于中文说明，不参与 `pip` 安装解析。
